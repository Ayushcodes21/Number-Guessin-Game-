# Number Guessing Game

A simple Python game where the user guesses a random number between 1 and 100 within a limited number of attempts.

## Features
- Random number generation
- Attempt limit
- User input validation
- Win/Loss conditions

## What I learned
- Python loops and conditionals
- Handling user input
- Basic game logic
